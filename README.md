# 📊 Telecom SQL Portfolio Project

## 🚀 Overview
This project simulates a telecom network inventory system used for analyzing device utilization and identifying decommission opportunities.

## 🧱 Data Model
- **Sites** – Location details
- **Devices** – Network equipment
- **Circuits** – Connectivity details
- **Decommission Recommendations** – Business decisions

## 📁 Project Structure

schema.sql → Table creation
queries.sql → Analysis queries
data/ → CSV datasets


## 💡 Key SQL Concepts Used
- Joins (INNER, LEFT)
- Aggregations (COUNT, SUM)
- Window Functions (RANK, DENSE_RANK)
- Subqueries
- Filtering & grouping

## 📊 Sample Business Questions Solved
- Which sites have the highest number of devices?
- Which devices are underutilized?
- Which devices should be decommissioned?
- Top devices by circuit usage

## 🛠️ How to Run
1. Run `schema.sql`
2. Load CSV files into tables
3. Execute queries from `queries.sql`
